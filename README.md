# InfoCell — Protótipo de site para venda de celulares e acessórios

Projeto de protótipo front-end desenvolvido para apresentar a estrutura,
organização e fluxo de um site de venda de celulares e acessórios
para uma loja física.

O foco deste projeto é demonstrar:
- organização das páginas
- disposição dos elementos
- estrutura HTML e CSS
- experiência do usuário (UX)

As imagens, produtos e conteúdos finais seriam adicionados
após validação do layout com o cliente.
